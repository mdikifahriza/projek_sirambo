SiRambo: Sistem Penyiraman Otomatis Berbasis Android

Inovasi berbasis ESP32 dan Firebase untuk solusi penyiraman tanaman modern.

Gambaran Umum Sistem

Sistem ini memadukan ESP32 sebagai pengontrol perangkat keras dengan aplikasi Android melalui Firebase untuk otomasi penuh dalam penyiraman tanaman.

Komponen Utama: ESP32

  • Sensor Kelembapan Tanah: Membaca kadar kelembapan tanah dan mengubahnya menjadi nilai persentase.
  • Sensor Suhu & Kelembapan Udara (DHT22): Mengukur suhu dan kelembapan lingkungan.
  • Pengontrol Relay: Mengaktifkan atau menonaktifkan pompa air secara manual atau otomatis berdasarkan pembacaan sensor.
  • Kontrol Suhu dan Kipas: Menyalakan fan atau heater berdasarkan suhu.
  • Kontrol LED: Memberikan indikasi status sistem secara visual.

Firebase: Jantung Penghubung

Firebase berfungsi sebagai pusat penyimpanan dan pengelolaan data, memungkinkan sinkronisasi real-time antara perangkat keras dan aplikasi.

  • Penyimpanan Data Sensor: Menyimpan data kelembapan tanah, suhu, dan kelembapan udara secara berkala.
  • Riwayat Sensor: Mencatat semua data historis untuk pemantauan jangka panjang.
  • Status Perangkat: Memperbarui status relay, LED, dan fan secara real-time.

Aplikasi Android: Kontrol di Genggaman

  • Monitoring Sensor: Menampilkan data real-time dari Firebase.
  • Pengaturan Threshold: Mengatur batas kelembapan dan suhu dari aplikasi.
  • Kontrol Manual & Otomatis: Mengaktifkan/deaktivasi relay, fan, dan LED secara langsung atau otomatis.
  • Pembaruan Status Otomatis: Sinkronisasi real-time tanpa perlu refresh manual.

Alur Kerja Sistem

  1. ESP32 membaca data dari sensor.
  2. Data dikirim ke Firebase secara berkala.
  3. Aplikasi Android membaca dan menampilkan data kepada pengguna.
  4. Pengguna dapat mengubah pengaturan atau mengontrol perangkat melalui aplikasi.
  5. ESP32 menerapkan perubahan yang diperbarui dari Firebase ke perangkat keras.

Kesimpulan

Dengan kombinasi ESP32, Firebase, dan aplikasi Android, SiRambo menawarkan sistem penyiraman tanaman yang sepenuhnya otomatis, fleksibel, dan mudah dikendalikan dari mana saja. Solusi modern untuk pertanian cerdas masa depan!
